# Car Price Prediction System

A Machine Learning project that predicts car prices based on various features using regression models.

This project helped me understand how numerical data can be used for prediction tasks and how different regression techniques perform on structured datasets.

## Concepts Used

* Data Preprocessing  
* Linear Regression  
* Lasso Regression  
* Feature Analysis  
* Scatter Plot Visualization  
* Train-Test Split  
* Regression Problem Understanding  

## Results

* Built Linear Regression and Lasso Regression models for car price prediction.  
* Compared performance of both models.  
* Observed that Lasso Regression helps reduce overfitting by penalizing less important features.  
* Visualized predictions using scatter plots.  

## What I Learned

Through this project, I understood how regression models are used to predict continuous values and how regularization (like Lasso) improves model performance in cases with multiple features.
